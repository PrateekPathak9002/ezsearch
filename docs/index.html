Search.js tutorial

Using search is extremely simple . Lets say you need to run a search with a input through a set of FAQ's on your website . For example purposes , my FAQs will just be a few p elements . 
<p class='myFAQs'> FAQ answer 1<p>
<p class='myFAQs'> FAQ answer 2<p>
<p class='myFAQs'> FAQ answer 3<p>
<p class='myFAQs'> FAQ answer 4<p>

Let's start by preparing these for search . Wrap all the elements you want to search through in a div with a unique id .

<div id='uniqueID1'>
	<p class='myFAQs'> FAQ answer 1<p>
	<p class='myFAQs'> FAQ answer 2<p>
	<p class='myFAQs'> FAQ answer 3<p>
	<p class='myFAQs'> FAQ answer 4<p>
<div >
Next add a input to your DOM. 
<input type='text' id='searchBar1'>
<div id='uniqueID1'>
	<p class='myFAQs'> FAQ answer 1<p>
	<p class='myFAQs'> FAQ answer 2<p>
	<p class='myFAQs'> FAQ answer 3<p>
	<p class='myFAQs'> FAQ answer 4<p>
<div >

That's it for preparing . Next import search if you haven't before .Since search works in javascript , add some script tags to your page to activate the search .Inside the script tags , call the following function anywhere. 

search(wrapperDIVid,searchElemClass,inputId)
The first argument should have an element , namely the wrapper div. In our case the id of the wrapper is uniqueID1. Therefore the first argument should be : 
document.getElementById('uniqueID1')

The searchElemClass argument should have the common id of the elements that have to be searched through. In our case the id is myFAQs .Therefore the second argument should be : 

document.querySelectorAll('.myFAQs')
Make sure to sure to use querySelectorAll only.

The third argument should be the input : 

document.getElementById('searchBar1')
The function now looks like this : 
search(document.getElementById('uniqueID1'),document.querySelectorAll('.myFAQs') , document.getElementById('searchBar1'))
That's it , you can now run it ! Example below .
