Search.js tutorial

Using search is extremely simple . Lets say you need to run a search with a input through a set of FAQ's on your website . For example purposes , my FAQs will just be a few p elements . 
<p class='myFAQs'> FAQ answer 1<p>
<p class='myFAQs'> FAQ answer 2<p>
<p class='myFAQs'> FAQ answer 3<p>
<p class='myFAQs'> FAQ answer 4<p>
Of course though , they can be anything . Next add a input to your DOM. 
<input type='text' id='searchBar1'>
<p class='myFAQs'> FAQ answer 1<p>
<p class='myFAQs'> FAQ answer 2<p>
<p class='myFAQs'> FAQ answer 3<p>
<p class='myFAQs'> FAQ answer 4<p>

All right ! You're halfway through ! Next import search using 

<script src="https://unpkg.com/ezsearch@1.0.1/search.js"> </script>
if you haven't before. Since search works in javascript , add some script tags to your page to activate the search .Inside the script tags , call the following function anywhere. 

search(searchElemClass,inputIdOrClass)

The searchElemClass argument should have the common class of the elements that have to be searched through. In our case the id is myFAQs .Therefore the first argument should be : 

'.myFAQs'

The second argument should be the id of the input : 

'searchBar1'
The function now looks like this : 
search('.myFAQs','#searchBar1')
That's it , you can now run it now ! Note that there is a third argument which are the options .Scroll below the example for the list of all options.
Remember though, if it doesn't work , check the console . ezsearch.js mostly always logs the errors .
